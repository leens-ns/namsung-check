남성초등학교 출결관리 시스템

사용 매뉴얼

메인 화면으로 공통 교사 방과후강사 관리자

공통

  1. 관리자에게 등록한 이메일과 동일한 Google 계정으로 로그인합니다. 로그인은 팝업이 아니라 Google 화면으로 안전하게 이동한 뒤 앱으로 돌아오는 방식입니다.
  2. 앱 설치로 홈 화면에 추가하고, 알림 대상자는 알림 켜기를 눌러 허용합니다.
  3. 공용 기기에서는 사용 후 반드시 로그아웃합니다.

공식 앱 주소는 namsung-check.firebaseapp.com입니다. 예전 GitHub 주소로 설치한 아이콘이 있다면 삭제하고 공식 주소에서 다시 홈 화면에 추가하세요.

학생 정보, 출결, 학부모 연락처를 촬영·메신저 전송·개인 기기 저장하지 마세요.

담임교사

  1. 출결 입력에서 자신의 학급과 학생 수를 확인합니다.
  2. 미입력 학생 모두 출석을 누른 뒤 결석·지각·조퇴 학생만 변경합니다.
  3. 색상으로 표시된 방과후 수강 학생은 상태를 한 번 더 확인합니다. 특이사항은 필요한 경우에만 작성합니다.
  4. 저장 전 확인에서 변경 명단을 검토하고 확정하고 저장을 누릅니다.

모든 입력 내용 저장됨이 표시되어야 서버 저장이 끝난 것입니다. 상태 버튼만 누른 단계는 저장 전입니다.

강사 배정도 함께 있는 계정은 사용자 정보의 사용 모드에서 담임교사·방과후강사 화면을 전환합니다.

학생 명단 변경

  • 자기 반 학생만 추가·수정·삭제할 수 있습니다.
  • 수정·삭제 내용은 즉시 공용 학생 명단에 저장되므로 관리자에게 별도로 전달할 필요가 없습니다.
  • 방과후 수강 요일과 부서는 선택 항목으로 입력하고, 수정·삭제 확인창에서 학생 정보를 다시 확인합니다.

출결은 날짜별로 보관됩니다. 새 운영일에는 미입력 상태로 시작하지만 이전 기록은 삭제되지 않습니다.

출결 조회

교사는 출결 조회에서 기본 선택된 자기 반의 일별 상세와 월별·학년도별 합계를 확인할 수 있습니다.

방과후강사

  1. 사용자 정보 아래 언어에서 한국어·영어·프랑스어·스페인어를 선택합니다. 매뉴얼과 알림도 선택 언어를 따릅니다.
  2. 출결 조회에서 자신의 담당 부서와 조회 날짜를 확인합니다.
  3. 수업 직전 최신 출결 새로고침을 누릅니다.
  4. 출석·결석과 특이사항을 확인해 수업에 반영합니다.
  5. 월별 또는 학년도별 탭에서는 담당 부서 학생의 기간 합계를 확인합니다.

교사가 입력한 결석·지각·조퇴는 방과후 화면에서 모두 결석으로 합산됩니다.

담임 배정도 함께 있는 계정은 사용자 정보의 사용 모드에서 두 화면을 전환할 수 있습니다.

강사는 조회만 가능하며 출결 입력, 학생 수정, 학부모 연락처 확인 권한이 없습니다.

관리자

학기 시작 설정 순서

  1. 예시 CSV를 내려받아 형식을 확인한 뒤 관리자 기기의 학생 CSV로 명단을 전체 교체합니다. CSV에 없는 기존 학생과 연락처는 삭제됩니다.
  2. 운영 요일, 학년별 최대 반 수, 요일별 방과후 부서 목록을 설정합니다.
  3. 담임교사 이메일과 학급을 단건·붙여넣기·CSV 중 편한 방식으로 배정합니다.
  4. 방과후강사 이메일과 담당 부서를 단건 또는 CSV로 등록합니다. 학교 계정도 강사로 명시 등록할 수 있습니다.
  5. 필요한 경우 추가 관리자를 지정합니다.
  6. 새 학기 강사 교체 시 강사 배정 전체 해제로 기존 강사 계정을 정리합니다.

같은 계정에 담임과 강사 배정을 함께 등록할 수 있습니다. 이 경우 로그인 후 사용 모드에서 담임교사·방과후강사 화면을 전환하며, 한쪽 배정을 삭제해도 다른 권한은 유지됩니다.

학년도는 해당 연도 3월 1일부터 다음 해 2월 마지막 날까지입니다. 관리자의 월별·학년도별 조회는 읽기 사용량을 줄이기 위해 부서를 하나 선택한 뒤 실행합니다.

CSV는 학생 명단의 최종 기준입니다. 교사가 개별 수정한 내용은 즉시 반영되지만, 이후 CSV를 다시 올리면 새 CSV 내용으로 덮어씁니다. 전체 삭제는 학생 명단과 연락처만 지우며 출결 기록은 유지합니다.

운영과 보안

  • 아침 입력, 교직원 재확인, 강사 확인 알림의 요일과 시간을 점검합니다.
  • 상단 미입력 숫자를 누르면 현재 범위의 미입력 학급 담임에게 알림을 요청하며, 다음 자동 알림 작업에서 전송됩니다.
  • 출결 조회는 담임 배정 관리자의 자기 반, 미배정 관리자의 방과후 수강학생 전체가 기본입니다. 학생 전체 조회는 보안 설정에서 필요한 기간에만 허용하며 비용 보호를 위해 일별 조회만 지원합니다.
  • 담당 변경·퇴직 계정은 즉시 삭제하거나 새 부서로 수정합니다.
  • 학부모 연락처는 필요한 순간에만 표시하고 확인 후 다시 비공개로 전환합니다.
  • 학생 CSV는 공개 링크가 아닌 관리자 기기의 파일만 사용합니다.
  • 새 학기에는 담임 배정 전체 해제로 기존 배정을 정리한 뒤 새 배정을 등록할 수 있습니다. 전체 해제는 관리자 권한과 강사 계정을 유지합니다.

무료 운영·자료 보관

  1. 관리자 설정 → 무료 운영·자료 보관에서 학교 보관 기준에 맞는 기간을 선택합니다.
  2. 오래된 출결 자동 삭제를 켠 뒤 보관 설정 저장을 누릅니다.
  3. Firebase 사용량 확인일을 정하면 해당 날짜부터 상단 안내와 관리자 알림이 표시됩니다. 안내나 사용량 확인 버튼을 누르면 모든 관리자 화면에서 사라지고 다음 달에 다시 표시됩니다.
  4. 상태 새로고침에서 자동 정리·웹앱 배포·출결 알림이 모두 초록색인지 확인합니다.
  5. GitHub 작업이 비활성화되어 있으면 해당 작업에서 Enable workflow를 눌러 다시 켭니다.

삭제된 출결은 복구할 수 없습니다. 자동 정리는 출결 기록만 대상으로 하며 학생 명단과 학부모 연락처는 유지합니다.
