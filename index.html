무료 방식

자동 삭제된 출결은 복구할 수 없습니다. 학생 명단과 연락처는 삭제하지 않으며, 한 번에 최대 10,000건만 정리합니다.

무료 기준: 저장 1GiB, 읽기 50,000건/일, 쓰기·삭제 각 20,000건/일. 한도를 넘기지 않았는지 월 1회 확인합니다.

월 1회 확인 방법
  1. Firebase 사용량 확인에서 저장 공간과 일일 읽기·쓰기·삭제가 무료 기준 미만인지 확인합니다.
  2. 배포·알림 전체 상태에서 최근 작업들이 초록색 성공인지 확인합니다.
  3. 이 화면의 마지막 실행일과 삭제 건수를 확인합니다.
  4. GitHub 작업이 비활성화되어 있으면 해당 작업에서 Enable workflow를 누릅니다.
